# my_bot wasm plugins

## wtest (Rust + ayiou-wasm-sdk)

The plugin author only implements regular Rust callbacks via `Plugin` trait.
ABI exports (`ayiou_alloc`, `ayiou_on_command`, `ayiou_on_regex`, `ayiou_on_cron`) are generated by SDK macro.
Callbacks return `Option<ReplyAction>`, so reply content is produced by wasm plugin code.

Source:
- `examples/my_bot/wasm/wtest_plugin_sdk/src/lib.rs`

Build (offline-friendly):

```bash
cargo build --offline --release \
  --target wasm32-unknown-unknown \
  --manifest-path examples/my_bot/wasm/wtest_plugin_sdk/Cargo.toml
cp examples/my_bot/wasm/wtest_plugin_sdk/target/wasm32-unknown-unknown/release/wtest_plugin_sdk.wasm \
  examples/my_bot/wasm/wtest_plugin.wasm
```

Runtime load path:
- `examples/my_bot/wasm/wtest_plugin.wasm`
