---
title: Plugin 宏
description: 使用 #[plugin] 构建默认插件入口
---

`#[plugin]` 是默认插件编写方式。它放在 `impl` 块上，impl 里的 async 方法会按方法名注册成命令，宏会生成 `RuntimePlugin` 实现，并默认提交到自动发现注册表。`Bot::run()` 会加载当前上下文类型匹配的自动发现插件。

高级场景仍然可以手写 `RuntimePlugin`，但默认文档只推荐这一套宏入口。

## 基本用法

```rust
use ayiou::{plugin, Context};

#[derive(Default)]
struct MyPlugin;

#[plugin(name = "my_plugin", prefix = "/")]
impl MyPlugin {
    async fn hello(&self, ctx: &Context) -> anyhow::Result<()> {
        ctx.reply_text("hello").await
    }
}
```

主入口不需要逐个注册宏插件：

```rust
ConsoleBot::console().run().await;
```

如果插件需要自定义构造参数，或只是测试/辅助插件，可以关闭自动注册：

```rust
#[plugin(name = "helper", prefix = "/", register = false)]
impl HelperPlugin {
    async fn help(&self, ctx: &Context) -> anyhow::Result<()> {
        ctx.reply_text("help").await
    }
}
```

## 可用属性

| 属性 | 说明 |
|---|---|
| `name` | 插件名，默认是结构体名的小写 |
| `description` | 插件描述 |
| `version` | 插件版本，默认 `0.1.0` |
| `prefix` | 插件级命令前缀，可重复声明 |
| `context` | 显式指定上下文类型，默认 `Context` |
| `register` | 是否自动注册，默认 `true`；关闭时写 `register = false` |

默认自动注册要求插件类型实现 `Default`，因为运行时会通过无参构造创建插件实例。

每个 async 方法默认注册为同名命令。例如 `async fn hello(...)` 会匹配 `/hello`。

## 参数解析

```rust
async fn ban(&self, ctx: &Context, user_id: i64, minutes: Option<u32>) -> anyhow::Result<()> {
    let minutes = minutes.unwrap_or(10);
    ctx.reply_text(format!("ban {user_id} for {minutes} minutes")).await
}
```

命令参数会按方法签名自动解析。支持任意 `FromStr` 类型、`Option<T>`、尾部 `String` 和尾部 `Vec<String>`。

## 平台扩展

默认上下文是 `Context`。通用插件优先通过 `Context` 的通用 API 写；确实需要平台专属 API 时，再通过 `ctx.extension::<...>()` 获取。

下面的 OneBot 类型需要依赖启用 `adapter-onebot-v11` feature。

```rust
use ayiou::adapter::onebot::v11::{ctx::Ctx, model::Message};
use ayiou::{plugin, Context};

#[derive(Default)]
struct WhoAmI;

#[plugin(name = "whoami", prefix = "/")]
impl WhoAmI {
    async fn whoami(&self, ctx: &Context) -> anyhow::Result<()> {
        if let Some(onebot) = ctx.extension::<Ctx>() {
            let login = onebot.get_login_info_data().await?;
            onebot
                .reply(Message::String(format!(
                    "bot={} user={}",
                    login.nickname,
                    onebot.user_id()
                )))
                .await?;
        }

        Ok(())
    }
}
```

如果平台上下文本身就是你的主入口，也可以指定 `context`：

```rust
#[derive(Default)]
struct OneBotOnlyPlugin;

#[plugin(name = "onebot-only", context = "ayiou::adapter::onebot::v11::ctx::Ctx")]
impl OneBotOnlyPlugin {
    async fn ping(&self, ctx: &Ctx) -> anyhow::Result<()> {
        ctx.reply_text("pong").await
    }
}
```

## 高级路径

需要声明非命令 handler、能力依赖、配置生命周期或复杂会话控制时，手写 `RuntimePlugin`，并从 `ayiou::core::plugin` 引入运行时类型。
