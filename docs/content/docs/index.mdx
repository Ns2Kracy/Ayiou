Ayiou 是一个面向 Rust Bot 场景的插件化框架。

- 支持清晰的运行时分层
- 支持 Driver / Adapter 解耦
- 支持从轻量插件到复杂后台插件的统一开发体验

## 核心能力

- `Driver`：连接 WebSocket / HTTP / Console 等底层输入输出
- `ProtocolAdapter`：适配 OneBot、Console 等协议
- `RuntimePlugin`：统一插件行为、生命周期、配置和事件处理
- `Bot` Runtime：负责插件注册、队列和事件分发；多渠道通过 adapter 接入同一个 Bot

## 插件开发

- `#[plugin] + #[command]`：默认插件编写入口，支持多命令插件与自动参数解析
- `HandlerDecl`：声明命令、正则、权限、优先级和 block 等执行约束
- OneBot v11 主动消息与上下文 API
- Console 适配器与本地调试体验
