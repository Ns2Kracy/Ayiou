---
title: Ayiou 框架
description: 一个面向多平台的 Rust Bot 框架（Driver + Adapter + Plugin）
---

Ayiou 是一个 Rust Bot 框架，核心目标是：

- 平台无关的插件开发体验
- 明确的 Driver / Adapter 分层
- 强类型命令系统与可扩展运行时能力

## 架构分层

- `Driver`：连接层（WebSocket/HTTP/Console 等）
- `ProtocolAdapter`：协议转换层（OneBot/Discord/...）
- `Bot` Runtime：分发事件、执行插件

## 当前能力

- `#[derive(Plugin)]`：快速单命令插件
- `#[bot_plugin] + #[command]`：多命令插件与自动参数解析
- OneBot v11 适配
- Console 适配（便于本地调试）
- 存储与会话抽象（开发者自定义后端实现）
- 轻量可观测（`MetricsSink` + 内存聚合 + 周期日志）
