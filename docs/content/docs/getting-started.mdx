## 安装依赖

```toml
[dependencies]
ayiou = { git = "https://github.com/Ns2Kracy/Ayiou.git" }
anyhow = "1.0"
tokio = { version = "1", features = ["full"] }
async-trait = "0.1"
```

## 核心概念

- `Bot<A>`：单 Bot 运行时 builder
- `Adapter / ProtocolAdapter`：把平台事件归一化到内核
- `RuntimePlugin`：处理消息、命令、生命周期、配置和主动发送
- `Context`：默认通用上下文

## 示例 1：OneBot v11

```rust
use ayiou::{command, plugin, Context, OneBotV11Bot};

#[derive(Default)]
struct EchoPlugin;

#[plugin(name = "echo", prefix = "/")]
impl EchoPlugin {
    #[command(name = "echo")]
    async fn echo(&self, ctx: &Context) -> anyhow::Result<()> {
        ctx.reply_text(format!("echo: {}", ctx.text())).await?;
        Ok(())
    }
}

#[tokio::main]
async fn main() {
    let bot = OneBotV11Bot::ws("ws://127.0.0.1:8080")
        .command_prefixes(["/", "!", "."])
        .with_plugin(EchoPlugin::default());

    bot.run().await;
}
```

## 示例 2：Console

```rust
use ayiou::{command, plugin, ConsoleBot, Context};

#[derive(Default)]
struct HelpPlugin;

#[plugin(name = "help", prefix = "/")]
impl HelpPlugin {
    #[command(name = "help")]
    async fn help(&self, ctx: &Context) -> anyhow::Result<()> {
        ctx.reply_text("Commands: /help").await
    }
}

#[tokio::main]
async fn main() {
    let bot = ConsoleBot::console().with_plugin(HelpPlugin::default());
    bot.run().await;
}
```

## 平台扩展

默认推荐直接使用 `Context`。当你需要平台原生 API 时，再通过扩展访问平台上下文。

Console 示例：

```rust
if let Some(console) = ctx.extension::<ConsoleCtx>() {
    let raw_line = console.line();
}
```

OneBot 示例：

```rust
if let Some(onebot) = ctx.extension::<Ctx>() {
    let login = onebot.get_login_info_data().await?;
}
```

## 高级插件

默认入口只推荐 `#[plugin] + #[command]`。如果需要非命令 handler、能力声明、配置生命周期或复杂会话控制，手写 `RuntimePlugin` 并从 `ayiou::core::plugin_system` 引入运行时类型。

## 常用 Context API

- `ctx.text() -> String`
- `ctx.user_id() -> String`
- `ctx.group_id() -> Option<String>`
- `ctx.reply_text("...")`
- `ctx.extension::<ConsoleCtx>()` / `ctx.extension::<Ctx>()`
