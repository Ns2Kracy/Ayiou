## 安装依赖

```toml
[dependencies]
ayiou = { git = "https://github.com/Ns2Kracy/Ayiou.git" }
anyhow = "1.0"
tokio = { version = "1", features = ["full"] }
async-trait = "0.1"
```

## 核心概念

- `Bot<A>`：单 Bot 运行时 builder
- `Adapter / ProtocolAdapter`：把平台事件归一化到内核
- `Plugin`：处理消息、命令和主动发送
- `Context`：默认通用上下文

## 示例 1：OneBot v11

```rust
use ayiou::adapter::onebot::v11::adapter::OneBotV11Adapter;
use ayiou::prelude::*;

#[derive(Plugin)]
#[plugin(name = "echo", command = "echo", prefix = "/")]
struct EchoPlugin;

impl EchoPlugin {
    async fn execute(&self, ctx: &Context) -> anyhow::Result<()> {
        let args = ctx.command_args().unwrap_or_default();
        if args.is_empty() {
            ctx.reply_text("Usage: /echo <text>").await?;
            return Ok(());
        }

        ctx.reply_text(format!("echo: {}", args)).await?;
        Ok(())
    }
}

#[tokio::main]
async fn main() {
    let bot = Bot::<OneBotV11Adapter>::new()
        .command_prefixes(["/", "!", "."])
        .register_plugin(EchoPlugin);

    bot.run(OneBotV11Adapter::new("ws://127.0.0.1:8080")).await;
}
```

## 示例 2：Console

```rust
use ayiou::prelude::*;

#[derive(Plugin)]
#[plugin(name = "help", command = "help", prefix = "/")]
struct HelpPlugin;

impl HelpPlugin {
    async fn execute(&self, ctx: &Context) -> anyhow::Result<()> {
        ctx.reply_text("Commands: /help").await
    }
}

#[tokio::main]
async fn main() {
    let bot = ConsoleBot::console().register_plugin(HelpPlugin);
    bot.run_stdio().await;
}
```

## 平台扩展

默认推荐直接使用 `Context`。当你需要平台原生 API 时，再通过扩展访问平台上下文。

Console 示例：

```rust
if let Some(console) = ctx.extension::<ConsoleCtx>() {
    let raw_line = console.line();
}
```

OneBot 示例：

```rust
if let Some(onebot) = ctx.extension::<Ctx>() {
    let login = onebot.get_login_info_data().await?;
}
```

## 自定义 start / handler

```rust
use ayiou::core::plugin_host::PluginHost;
use ayiou::prelude::*;

#[derive(Plugin)]
#[plugin(
    name = "ticker",
    command = "tick",
    start = "bootstrap",
    handler = "dispatch"
)]
struct TickerPlugin;

impl TickerPlugin {
    async fn bootstrap(&self, _host: PluginHost<Context>) -> anyhow::Result<()> {
        Ok(())
    }

    async fn dispatch(&self, ctx: &Context) -> anyhow::Result<bool> {
        ctx.reply_text("tick").await?;
        Ok(true)
    }
}
```

## 常用 Context API

- `ctx.text() -> String`
- `ctx.user_id() -> String`
- `ctx.group_id() -> Option<String>`
- `ctx.reply_text("...")`
- `ctx.command_name()` / `ctx.command_args()`
- `ctx.extension::<ConsoleCtx>()` / `ctx.extension::<Ctx>()`
