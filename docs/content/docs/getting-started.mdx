## 安装依赖

```toml
[dependencies]
ayiou = { git = "https://github.com/Ns2Kracy/Ayiou.git" }
anyhow = "1.0"
tokio = { version = "1", features = ["full"] }
async-trait = "0.1"
```

## 核心概念

- `Bot<A>`：Bot 运行时
- `Driver`：底层连接驱动
- `Adapter / ProtocolAdapter`：协议适配层
- `Plugin`：消息处理插件

## 示例 1：OneBot v11

```rust
use ayiou::adapter::onebot::v11::adapter::OneBotV11Adapter;
use ayiou::prelude::*;

#[derive(Plugin)]
#[plugin(name = "echo", command = "echo", prefix = "/")]
struct EchoPlugin;

impl EchoPlugin {
    async fn execute(&self, ctx: &Ctx) -> anyhow::Result<()> {
        let args = ctx.command_args().unwrap_or_default();
        if args.is_empty() {
            ctx.reply_text("Usage: /echo <text>").await?;
            return Ok(());
        }
        ctx.reply_text(format!("echo: {}", args)).await?;
        Ok(())
    }
}

#[tokio::main]
async fn main() {
    let bot = Bot::<OneBotV11Adapter>::new()
        .command_prefixes(["/", "!", "."])
        .register_plugin(EchoPlugin);

    bot.run(OneBotV11Adapter::new("ws://127.0.0.1:8080")).await;
}
```

## 示例 2：Console

```rust
use ayiou::prelude::*;

#[derive(Plugin)]
#[plugin(
    name = "help",
    command = "help",
    prefix = "/",
    context = "ayiou::adapter::console::ctx::Ctx"
)]
struct HelpPlugin;

impl HelpPlugin {
    async fn execute(&self, ctx: &ConsoleCtx) -> anyhow::Result<()> {
        ctx.reply_text("Commands: /help").await
    }
}

#[tokio::main]
async fn main() {
    let bot = ConsoleBot::console().register_plugin(HelpPlugin);
    bot.run_stdio().await;
}
```

## 带状态插件

如果你的插件不是 unit struct，而是需要自己的字段、构造函数或初始化逻辑，注册时直接传实例即可，例如：

```rust
use ayiou::prelude::*;
use ayiou_plugin_bilibili_live::BilibiliLivePlugin;

let bot = Bot::<ayiou::adapter::onebot::v11::adapter::OneBotV11Adapter>::new()
    .register_plugin(BilibiliLivePlugin::new());
```

## 进阶：自定义 start / handler

`#[derive(Plugin)]` 默认会调用 `execute(&self, ctx)`。如果插件需要在启动阶段注册定时任务，或者想自己控制 `handle()` 的返回值，也可以这样写：

```rust
use ayiou::core::plugin_host::PluginHost;
use ayiou::prelude::*;

#[derive(Plugin)]
#[plugin(
    name = "ticker",
    command = "tick",
    start = "bootstrap",
    handler = "dispatch"
)]
struct TickerPlugin;

impl TickerPlugin {
    async fn bootstrap(&self, _host: PluginHost<Ctx>) -> anyhow::Result<()> {
        Ok(())
    }

    async fn dispatch(&self, ctx: &Ctx) -> anyhow::Result<bool> {
        ctx.reply_text("tick").await?;
        Ok(true)
    }
}
```

更完整的说明见“Plugin 宏”。

## 常用 Context API

- `ctx.text() -> String`
- `ctx.user_id() -> i64`
- `ctx.group_id() -> Option<i64>`
- `ctx.reply_text("...")`
- `ctx.command_name()` / `ctx.command_args()`
